Source code Struktur Data dasar berjudul "Program Penilaian Mahasiswa"
diprogram dengan menggunakan bahasa pemrograman C++. Pada source code program
ini tidak digunakan algoritma khusus, tetapi dari hasil pemikiran pemrogram
sendiri. Dalam program ini terdapat 7 menu program utama yakni :

1. Tambah Mahasiswa    -> Untuk menambah data mahasiswa baru
2. Edit Data           -> Untuk menyunting data mahasiswa
3. Hapus Data          -> Untuk menghapus data mahasiswa
4. Urutkan Data	       -> Untuk mengurutkan data (unworked)
5. Cari Data           -> Untuk mencari data (unworked)
6. View Hasil          -> Untuk menampilkan hasil data mahasiswa
7. Tutup Program       -> Untuk mengakhiri program

Pesan pemrogram (saya), source code ini saya rasa mungkin masih sangat 
sederhana dan kurang sempurna, contohnya saja pada menu Sorting dan
Searching tidak saya fungsikan, karena saat pembuatan Projectnya
saya kurang paham mengenai itu tadi, maka dari itu alangkah baiknya 
apabila anda yang akan menjadi user program saya ini berkenan mempelajari 
alur daripada algoritma program saya ini guna melakukan peng-Upgrade-an
